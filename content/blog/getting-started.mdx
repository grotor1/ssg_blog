# Getting Started with Nuxt and MDX

This is my first blog post using Nuxt Content with MDX support.

## Why Nuxt Content?

Nuxt Content is a powerful file-based CMS that makes it easy to write and manage content:

- Write in Markdown or MDX
- Query content with a MongoDB-like API
- Auto-generate navigation and table of contents
- Vue components in your content
